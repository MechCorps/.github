# MechCorps

**Ever wanted to pilot a 3-story tall, 75-ton walking battle mech armed to the teeth with lasers, cannons, and missiles?**

[MechCorps Entertainment](http://www.mechcorps.com) operates **Virtual World BattleTech Cockpit Simulator Pods** — arcade-grade giant robot simulators where players battle other pilots head-to-head, in teams, or against AI opponents in a virtual world of futuristic combat. Based in Houston, TX (co-located with [Battlefield Houston](http://www.mechcorps.com/contact-us)), MechCorps also tours conventions across the country, bringing the BattleTech experience to fans everywhere.

## 🤖 What is MechCorps?

MechCorps runs **Virtual World Tesla II BattleTech Cockpit Simulator Pods** — full-immersion, networked mech combat simulators. Players strap into a cockpit, choose their BattleMech, and go to war against live opponents. After each battle, pilots review a replay and receive a scoresheet to settle the bragging rights.

MechCorps hosts:

- **Drop-in sessions** at their Houston HQ
- **BattleTech Boot Camps** — training events for new pilots
- **SlagFest** and special event weekends
- A **nationwide convention tour** bringing the pods to events like Adepticon, Comicpalooza, Nan Desu Kan, DragonCon, and more

## 🛠️ This GitHub Organization

This GitHub organization is the **development arm of MechCorps**. Our goal is to develop new software solutions that keep the simulators running reliably for the long term — ensuring the community can keep playing for years to come.

That means:

- Maintaining and improving the software that powers the simulator pods
- Building new tools and integrations to extend simulator capabilities
- Ensuring long-term compatibility as hardware and software ecosystems evolve
